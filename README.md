# 집회 시위 제대로 아이폰
정치 사상의 자유마저도 결사 표현의 자유마저도  
아아 권력에 자본과 독점에 난자당한 이 산하에  
구현하리라 민중의 새 어플 집회시위제대로!  

## 앱 지원 범위
* iPhone4 이상
* iOS7 이상

## 빌드 환경 설정
이 어플리케이션은 라이브러리 의존성 관리를 위해 cocoapods을 사용합니다.  
clone 후 아래 명렁어를 실행해 주십시오.  
[cocoapods 설정이 안되었을 경우]  
> sudo gem install cocoapods  
> pod setup  

[공통]
해당 프로젝트 루트로 간 후  
> pod install
